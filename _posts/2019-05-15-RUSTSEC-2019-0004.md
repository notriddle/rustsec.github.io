### Description

Affected versions of this crate did not properly verify ed25519 signatures.
Any signature with a correct length was considered valid.

This allows an attacker to impersonate any node identity.

### Patched Versions

- `>= 0.7.1`
- `>= 0.8.1`

### Unaffected Versions

- `< 0.3`
